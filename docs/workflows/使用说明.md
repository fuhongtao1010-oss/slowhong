# A股趋势分析工具使用说明

## 启动工具

1. 打开终端
2. 进入项目目录: `cd "/home/hong/文档/脑子/08 工作流/A股趋势分析工具"`
3. 激活虚拟环境: `source venv/bin/activate`
4. 启动应用: `python -m flask run --port 5001`

## 使用方法

2. 在输入框中输入股票代码（例如：000001，600519，300750等）
3. 点击"分析"按钮查看结果

## 功能说明

- 趋势评分: 根据股票价格与移动平均线的关系计算出的评分(0-100分)
- 趋势标签: 根据评分生成的趋势判断(短期偏强、震荡整理、偏弱等)

## 停止工具

在终端中按 Ctrl+C 组合键停止应用，然后输入 `deactivate` 退出虚拟环境。
