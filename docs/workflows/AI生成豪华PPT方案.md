# AI 生成豪华 PPT 的几种方案

> 来源: https://x.com/gorden_sun/status/2032141698562998332
> 作者: @gorden_sun (谷登)

## 方案一：用 Nano Banana Pro/Banana 2 生成 PPT 图片

生成高质量、信息密集的幻灯片图片

- **优点**: 质量高、图形复杂、可多图
- **缺点**: 中文字可能有错误、难以编辑
- **产品**: NotebookLM、Youmind、定制 Banana API 产品

## 方案二：用 AI Agents 编辑 PPT 文件

使用 Claude Cowork、Codex 等工具编辑 PPT 模板

- **优点**: 成本低、直接可编辑
- **缺点**: 限于模板设计、从零创建困难

## 方案三：通过 SVG 网页生成 PPT

创建带动画的网页演示

- **优点**: 动画流畅、视觉效果好
- **缺点**: 导出 PPT 后动画丢失、设计较简单
