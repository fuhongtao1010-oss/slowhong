# A股趋势分析工具项目总结

## 项目文件

1. app.py - 主应用文件
2. templates/index.html - 前端界面
3. requirements.txt - 依赖文件
4. .env - 环境变量配置文件
5. 使用说明.md - 详细的使用说明
6. 项目总结.md - 项目总结

## 使用方法

1. 安装依赖: `pip install -r requirements.txt`
2. 配置环境变量: 在`.env`文件中设置`MINIMAX_API_KEY`
3. 运行应用: `python app.py`
4. 在浏览器中访问: `http://localhost:5000`

## 功能特点

1. 支持多只股票同时分析
2. 集成AKShare获取实时行情数据
3. 计算并显示移动平均线等技术指标
4. 生成趋势评分和标签
5. 提供Web界面供用户输入股票代码并查看分析结果
