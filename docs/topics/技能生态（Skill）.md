# Skill Ecosystem

## 核心问题
- 为什么近期 AI 社区会快速出现大量以 “skill” 为单位传播的项目？
- Skill 到底是功能封装、工作流模板，还是人格 / 方法论的可复用接口？
- 哪些 Skill 只是热点话题，哪些会沉淀为长期工具？

## 子主题
- 人格蒸馏（persona distillation）
- 名人思维模型 Skill
- 内容创作与多媒体 Skill
- 聊天软件 / agent 接入型 Skill

## 当前理解
- Skill 热潮的关键并不只是“让 AI 会做一件事”，而是把人格、风格、经验、流程和思维框架包装成可重复调用的模块。
- 传播最强的 Skill 往往同时满足三个条件：
  - 容易理解
  - 容易代入
  - 容易想象自己的使用场景
- 从长期价值看，基础设施类与内容生产类 Skill 更可能稳定沉淀；纯噱头型人格模仿项目则更容易快速过热后降温。
