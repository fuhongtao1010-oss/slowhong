# Persona Distillation

## 核心问题
- 如何把一个人的表达风格、思维方式、经验结构蒸馏成可交互的 Skill？
- 人格蒸馏的价值究竟在效率、陪伴、知识保留还是身份模拟？
- 这种蒸馏容易停留在语气模仿，还是能够逼近认知结构？

## 子主题
- 同事 / 老板 / 前任 / 自己 等贴身人物蒸馏
- 博主 / 公众人物的风格迁移
- 数字永生 / 第二大脑
- 人格蒸馏的伦理边界与误差来源

## 当前理解
- 人格蒸馏是这一波 Skill 热潮中最有传播力的方向之一，因为它把抽象的 AI 能力变成了“某个人在和你说话”。
- 用户追求的不只是说话语气，而是某种可持续调用的角色感、风格稳定性和经验压缩。
- 真正困难的部分通常不是“像不像”，而是：
  - 数据来源是否足够
  - 风格是否稳定
  - 价值观与判断是否一致
  - 是否出现误导性拟真
