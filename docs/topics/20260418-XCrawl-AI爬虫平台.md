# XCrawl - AI网页爬虫API平台

## 是什么

AI驱动的网页抓取API平台，将任何网站转换为结构化数据。

## 核心产品

- **Scrape API** — 单次API请求提取JSON/Markdown/截图
- **Search API** — 收集Google等搜索引擎SERP数据
- **Crawl API** — 智能爬取多页面网站
- **Map API** — 导出域名所有可发现URL
- **SERP API** — 获取搜索结果排名和摘要

## 核心优势

- LLM-ready JSON 和 clean Markdown 输出
- 自动轮换全球住宅代理
- 99%+ 数据提取成功率
- 先进浏览器指纹和反爬虫保护

## 应用场景

- 生成式AI和LLM训练数据
- B2B潜在客户生成
- 竞品情报监控
- 价格监控和动态定价
- SEO监测和SERP分析
- AI代理自动化

## 集成

- MCP Server（支持Claude等AI助手）
- n8n集成
- Zapier/Make无代码自动化
- 官方SDK：Python、Node.js
