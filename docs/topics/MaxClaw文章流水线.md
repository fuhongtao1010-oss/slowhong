手把手教你小龙虾（maxclaw）用搭建文章生产流水线

我知道很多在观望小龙虾的朋友，想尝鲜卡在最重要的一点上，就是部署小龙虾。

配置小龙虾过程极为繁琐，好不容易配置好了，但发现真正想随时随地使用，还得配置一下飞书等通讯工具。

不管付费还是折腾，一连串部署下来，最后看着聊天界面发呆。

为什么？因为部署好了，却没有一点生产力。

昨天我配置MiniMax API 的时候，发现他们推出了个有手就能行的配置方案。动动手就能配置好，小龙虾和手机聊天功能。更牛逼的是，他内置几套有生产力的小龙虾配置模板。

今天我就给大家演示如何利用这个模板造出你的7*24 小时文章生产流水线。

MaxClaw 有国内和国外两个版本，我选择国外的版本。因为他们国内版本爆火，可能会出现资源不足的问题。我就配置国外的版本。

登录国外官网：

按照下图步骤找到配置界面

[![Image 2: Image](https://pbs.twimg.com/media/HC8FCCSawAAoxno?format=jpg&name=small)](https://x.com/lxfater/article/2030848999742398525/media/2030847489499381760)

然后按照下图步骤，点击就部署好了

[![Image 3: Image](https://pbs.twimg.com/media/HC8GFX-aEAAgxB0?format=jpg&name=small)](https://x.com/lxfater/article/2030848999742398525/media/2030848646372265984)

没错，就是这么简单，动动小手就行，毕竟是花钱的服务。

OpenClaw 没接入通讯工具发挥不了生产力的，但配置飞书等通讯工具对普通用户来说，很麻烦。但官网直接将通讯功能集成到 MiniMax 的手机端就能用了，不需要去配置了。

在手机商店搜索 MiniMax，下载安装，侧边栏找到 MaxClaw 就能用了。具体如下图所示。

[![Image 4: Image](https://pbs.twimg.com/media/HC8FIjcaYAArwJQ?format=jpg&name=medium)](https://x.com/lxfater/article/2030848999742398525/media/2030847601478885376)

正常流程，安装小龙虾后，还要配大模型 API，生图 API，生视频的 API，联网的 API。但官方一下子默认都给你配置完毕了，无需配置，你就能拥有了小龙虾所有的功能。

能生图，生视频，联网搜索，但这些都是基本功能，我下面要给你展示如何一键将小龙虾变成内容机器。

现在 AI 大幅度提高了生产力，未来供给端竞争将会越来越激烈。但是全中国人口有限，大家的注意力有限，未来无法抢夺到注意力的人，就算有产品也很难销售出去。

为了争夺注意力，我们必须构建出一条内容生产流水线来获取注意力。

今天将会给大家演示一下，如何一键 构建出生成下面文章的 MaxClaw Agent。

没有做过自媒体的朋友可能看不出来，但是这里面基本涵盖大部分自媒体从业者的写作流程了。

下面为大家慢慢演示，逐步看看各个环节的效果如何！！

第一步，自动收集资料，你看下图，他框框收集一大堆的网页。传统流程个人去看，不得看半天。更有意思的是，他好像知道这种内容有点多了，叫我选择一个写作角度。

大家可能不懂写作角度是什么，但是你只要知道这个东西决定了大部分文章能不能火。当 AI 开始有意识讨论这个话题的时候，说明它已经入了写作的门了。
