# Skill · creao-ecommerce-design

## 简介

基于 Creao 6角色协同电商视觉设计系统的**设计规范文档版** Skill。输出内容为 7 模块完整**文字设计方案**（品牌策略、视觉规范、卖点文案、场景描述、规格说明等），而非直接可用的生图 prompt。

## 核心能力

用户报产品名 → 输出完整 7 模块设计规范文档 → 询问是否存脑子

## 适用输出物

- 品牌策略简报（品牌关键词、色彩方案、视觉风格）
- 7 模块设计规范（每个模块的布局、色值、字体、文案）
- 卖点卡片文案
- 场景应用卡片内容
- 规格与物流模块文案
- 行动召唤文案

## Skill 路径

`/home/hong/.hermes/skills/productivity/creao-ecommerce-design/SKILL.md`

## 使用场景

用户需要：
- 完整的设计方案文档供设计师参考
- 中英文双语的设计规范
- 需要了解每个模块的视觉布局和配色
- 后续由设计师手动制作图片

## 与生图 Prompt 版的区别

| | creao-ecommerce-design（规范版） | ecommerce-product-image-prompts（prompt版） |
|---|---|---|
| 输出物 | 文字设计规范文档 | Midjourney/SD/DALL-E 生图 prompt |
| 用途 | 设计师参考 | 直接喂给 AI 生图工具 |
| 包含内容 | 配色/字体/布局/文案全套 | 英文摄影描述，只给 prompt |
| 后续步骤 | 设计师按规范制作 | 出图后需自己排版加文字 |
