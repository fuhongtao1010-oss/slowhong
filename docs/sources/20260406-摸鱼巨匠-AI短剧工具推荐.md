# 2026-04-06 - 摸鱼巨匠 - AI短剧工具推荐

## 摘要
这是一条 AI 短剧工具推荐帖，重点列出 3 个更聚焦、更成熟的项目：Moyin Creator、waoowaoo、huobao-drama。相比广撒网式的工具盘点，这条更像是精选推荐，强调的是“剧本到成片”的完整自动化能力。

## 原文整理
最近 AI 短剧太火了！快收藏这几个 GitHub 项目做第一波吃螃蟹的人！

1. 魔因漫创 Moyin Creator
- 桌面端 AI 影视生产级工具
- 深度绑定 Seedance 2.0
- 实现「剧本 → 角色 → 场景 → 分镜 → 成片」的全流程批量化生产
- 特别适合需要高效率、多镜头连贯输出的创作者
- 仓库： https://github.com/MemeCalculate/moyin-creator

2. waoowaoo AI 影视 Studio
- 工业级全流程 AI 影视生产平台
- 专攻“网文 / 小说直转视频”
- 从文本自动解析角色、场景、剧情
- 一键生成带配音的完整短剧 / 漫画视频
- 仓库： https://github.com/saturndec/waoowaoo

3. huobao-drama
- 极简一站式 AI 短剧生成平台
- 主打「一句话 / 一个主题 / 一个剧本 → 完整成片」
- 全自动化程度高，对新手友好
- 自动完成：剧本改写、角色提取、分镜拆解、生图、视频生成、TTS 配音、FFmpeg 合成、字幕与导出
- 仓库： https://github.com/chatfire-AI/huobao-drama

## 关键信息
- 这条帖子聚焦在 3 个代表性强的项目上，强调的是“短剧生产闭环”。
- 三个项目分别代表：
  - 批量化生产与工业级工作流：Moyin Creator
  - 小说直转视频：waoowaoo
  - 一句话到成片、极简自动化：huobao-drama
- 如果目标是先试跑，3 个项目已经足够形成一个很好的起点对比。
