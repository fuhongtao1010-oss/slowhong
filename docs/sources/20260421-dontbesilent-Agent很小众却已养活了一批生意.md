## 正文

Agent 很小众，却已经养活了一批生意

已知：以 claude code/codex 为首的 Agent 不可能普及

但是：用 claude code 的人天天发内容到社交媒体平台，看的人会有焦虑

于是：会催生一个市场，这个市场规模和 claude code/codex 用户数并不直接相关，应该说是和平台上相关的内容流量呈非线性相关

那么：这里就会出现各种各样的供需组合，你可以看到这些生意发展成不同的叙事

比如说会有人提供远程或上门代安装服务，但是这个服务的买家呢，常常并不会深度使用这些产品，所以卖家基本上是拿时间换钱。可是你会看到有些卖家提供了这个服务之后，会把自己这段时间上门安装所观察到的情况，包装成一个创业履历，发成自媒体 vlog。这可能就是一个博主很好的起点

还有一些人选择在这个环节做中转站。考虑到我们这一波机会和上一波移动互联网不太一样，并不能因为 LLM 的出现而有百花齐放的应用创新（内部自用的不算）。反而呈现出 OpenAI 和 Anthropic 吞噬一切的趋势。用 API 帮助大家用上这些明星产品，就是一个好的选择。可是这个生意实在是太暴利，太好赚钱。能把这个生意合规化的选手并没有太多。这个生意的性质非常接近于算命。金额小没人管你，金额大的话，出事只是时间问题

但是如果中转站服务对接给企业在后面做扶持呢？那又发展成另外一回事。甚至也许这只是和企业们开始沟通的一把钥匙。哪怕后面 API 服务甩给别人来供货都可以

上面说的这三个可能是在 X 上面的用户会容易选择的路径。但你转头看向国内的话，就会发现，大家喜欢在「低价课程」和「几千几万十几万的一键爆款智能体」之间做二选一

前者（低价课）严格来讲不算是个真正的产品。它是一个通往更高价的产品的一把钥匙。那个更高价的产品是咨询还是陪跑，还是做一些落地的服务，就看个人本事了

后者（割韭菜智能体）表面上看起来是一个很好的生意，但是如果你近距离接触过，知道他们退款退了多少，知道他们给抖音交了多少广告费的话，可能你会有不一样的看法。但这还不是更重要的。更重要的是，这种产品以一种极度贴合人性的姿态存在，而无法介入到真实的业务杠杆里。这意味着最靠后，最有深度的那些真正值得赚的钱（比如介入到真实的企业主的业务里，有合作有投资）完全无法通过这个破智能体打开 —— 因为你的买家实在是太菜了

我自己目前打开了 1.5 个窗口。第一个窗口是最近两场线下课都破了 100 人。我从来没有用 Agent 这个事情贩卖过焦虑（当然，不排除有人感受到了焦虑，从而拉黑了我😂）。我说的都是真实的，我用 Agent 辅助到我自己业务的流程和案例。但与此同时呢，我也知道，一定是有人出于自己的焦虑而来参加了。考虑到，我并不认为我是一个贩卖焦虑的存在，且 Agent 在我这是手段而不是目的，所以这种增长我也就欣然接受吧 😂

另外 0.5 个窗口，现在不好意思说，是我想做一个 toC 的工具站。现在八字还没一撇，事情还没干成就不过多吹嘘。思路上呢，这并不是一个 AI Native 的产品，还是传统互联网产品。就是一个自媒体工具包，仅此而已。只不过如果没有大模型的话，这个产品肯定无法存在。codex 正在开发中 ……
