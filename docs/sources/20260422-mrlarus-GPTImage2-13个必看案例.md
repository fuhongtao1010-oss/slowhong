# GPT-Image-2 全量开放！狂刷几百张后，我整理了13个必看案例

## 摘要

**GPT-Image-2 亮点：**
- 中文渲染新突破——课本、报纸、药方、书法都能生成
- 懂内容的设计师——一句描述，缺的信息自己补全
- 支持极简 prompt
- 可追问微调，不满意自动修改

**背后是 Agent** — 能理解意图、搜索内容、整理组合、策划设计风格、合理排版，最终渲染成图。

## 13个必看案例

### 一、图鉴式拆解图（展板级别）

国家博物馆展板级质感，结构拆解，中文标注、材质说明、纹样寓意、色彩含义一应俱全。

**通用模板：**
```
请根据【主题】自动生成一张"博物馆图鉴式中文拆解信息图"。

要求整张图兼具真实写实主视觉、结构拆解，中文标注、材质说明、纹样寓意、色彩含义和核心特征总结。

整体风格应为：国家博物馆展板，历史服饰图鉴，文博专题信息图，而不是普通海报、古风写真、电商详情页或动漫插画。

避免：海报感、影楼感、电商感、动漫感、cosplay感、乱标注、错结构、糊字、假材质、过度装饰。
```

---

### 二、古文真迹 + 书法临摹帖

1. **还原千古名篇**：兰亭集序、出师表、滕王阁序、岳阳楼记……惟妙惟肖，满页密集中文书法没有崩坏！
   - 提示词：`生成【名篇名称】真迹图片`

2. **进阶版（注入情绪）**：AI 在理解汉语文学里的情绪，笔墨变化更丰富
   - 提示词：`生成【名篇名称】真迹图片，请结合作品情感核进行生图`

3. **摹字帖版（不同字体风格）**
   - 提示词：`生成一张【字体】书法临摹字帖`

---

### 三、轮廓宇宙图（收藏版叙事海报）

用巨大的人物/器物/建筑侧脸剪影作为外轮廓，内部自然生长出完整的世界观、场景、符号和叙事，纸张颗粒、边缘飞白、水彩刷痕、空气透视……一眼看过去就像收藏版电影海报或设计师作品。

**通用模板：**
```
请根据【主题】自动生成一张高审美的"轮廓宇宙/收藏版叙事海报"风格作品。

不要将画面局限于固定器物或常见容器，优先选择最有象征性、最有空间感、最能体现史诗感、神秘感或设计感的轮廓形式。

画面要求：
- 第一眼有强烈的主题识别度和轮廓记忆点
- 第二眼有完整丰富的叙事世界
- 第三眼仍有细节和余味

不要普通背景拼接，不要生硬裁切，不要模板化奇幻素材，不要游戏宣传图感，不要过度卡通化。
```

---

### 四、科普百科图

一张图把动物、植物、历史、科学等知识整理清楚，模块化排版、图鉴感、百科感拉满。

**通用模板：**
```
请根据【主题】生成一张高质量竖版「科普百科图」。

包含：主题主视觉 + 局部特征放大细节 + 圆角模块化信息分区 + 标题层级与重点标签 + 百科内容 + 可视化评分/要点总结

内容栏目自动适配：基础档案、分类信息、外观特征、习性/生态、形成机制、养护建议、风险注意事项、优缺点对比、快速评分卡。
```

---

### 五、电商详情图

颜色、光影、材质、卖点排版、场景氛围全部拉满，直接能拿来用或参考。

**通用模板：** `生成【产品名称】电商详情图`

---

### 六、健身信息图

输入想练的部位或动作，自动输出完整专业的训练计划：热身、主训动作（组数×次数+RIR+要点+常见错误）、进阶路径、替代动作、恢复建议。

**通用模板：**
```
请生成一张中文健身信息图，主题为：【xxx】。

默认训练目标为"增肌 + 基础力量提升"，默认训练水平为"新手到中级"，默认单次训练时长40–60分钟。

必须包含：标题区、训练目标区、热身区、主训练区（4–6个动作）、进阶/解锁逻辑区、替代动作区、执行提醒区、恢复建议区。
```

---

### 七、高考真题试卷

卷面排版、字体、插图、公式渲染全都极其真实，肉眼几乎分不出是 AI 还是扫描件。

**通用模板：** `生成2026年高考【科目名称】真题试卷图`

---

### 八、旅游攻略信息

仅用一句话出一张完整、清晰、美观的3天旅游攻略，包含每日行程、景点推荐、路线安排、餐饮tips、实用建议等。

**通用模板：** `生成【城市】三天/五一旅游攻略图`

---

**通用模板：** `生成【古人姓名】在【平台名称】上的截图`

### 十、游戏实机截图

光影、UI、动作、场景氛围全部还原得像真的一样。

**通用模板：** `生成GTA6【画面主题内容】实机截图`

### 十一、人物关系图

把动漫、小说、历史里错综复杂的人物关系，用颜色、线条、层级一图呈现，核心人物突出、关系类型一目了然。

**通用模板：**
```
请根据【主题】，生成一张高设计感的人物关系图海报。

自动完成：
- 提取最具代表性的 6–12 位关键人物
- 识别并展示关键人物关系（血缘、爱情、友情、联盟、敌对、师徒等）
- 用不同颜色、线型、箭头区分不同关系
- 核心人物最突出，形成明确视觉层级
```

---

### 十二、密集中文排版

满页文字、复杂排版（日报、菜单、课本、老黄历）都能生成清晰规整、几乎无乱码。

**通用模板：** `生成【场景/内容】图片`

---

### 十三、中西医手写药方

极简提示词渲染出手写体、纸张材质、印章、处方场景，甚至主动把姓名、印章等敏感信息打码。

**通用模板：** `生成一张手写中/西医药方图`
