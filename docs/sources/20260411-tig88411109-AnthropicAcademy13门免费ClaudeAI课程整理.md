## 课程清单（短链接已展开）
1. Claude 101
   - 说明：学习如何将 Claude 用于日常工作任务，了解核心功能，并探索更多高级学习资源。适合初学者快速上手 Claude。
   - 链接：https://anthropic.skilljar.com/claude-101

2. AI Fluency: Frameworks & Foundations
   - 说明：掌握与 AI 系统高效、伦理、安全协作的核心框架和基础知识，适合所有人提升 AI 素养。
   - 链接：https://anthropic.skilljar.com/ai-fluency-framework-foundations

3. Introduction to Agent Skills
   - 说明：学习与 Claude 一起处理真实文件和项目，涵盖 Cowork 任务循环、插件技能、文件/研究工作流及多步项目管理，让你第一周就能高效生产。
   - 链接：https://anthropic.skilljar.com/introduction-to-claude-cowork

4. Building with the Claude API
   - 说明：全面掌握使用 Claude API 构建应用的完整流程，包括模型调用、最佳实践和开发技巧，适合开发者。
   - 链接：https://anthropic.skilljar.com/claude-with-the-anthropic-api

5. Claude Code in Action
   - 说明：将 Claude Code 集成到开发工作流中，学习文件读取、命令执行、代码修改、上下文管理和自定义工作流等实战技能。
   - 链接：https://anthropic.skilljar.com/claude-code-in-action

6. Intro to Model Context Protocol
   - 说明：从零开始用 Python 构建 Model Context Protocol（MCP）服务器和客户端，掌握 tools、resources、prompts 三大核心原语，实现 Claude 与外部服务连接。
   - 链接：https://anthropic.skilljar.com/introduction-to-model-context-protocol

7. MCP: Advanced Topics
   - 说明：深入学习 MCP 高级实现模式，包括采样、通知、文件系统访问、传输机制等，适用于生产级 MCP 服务器开发。
   - 链接：https://anthropic.skilljar.com/model-context-protocol-advanced-topics

8. AI Fluency for Students
   - 说明：专为学生设计，帮助你在学习、职业规划和学术中安全高效地使用 AI，提升个人竞争力。
   - 链接：https://anthropic.skilljar.com/ai-fluency-for-students

9. AI Fluency for Educators
   - 说明：专为教师、课程设计师和教育领导者设计，将 AI Fluency 应用到教学实践和机构策略中。
   - 链接：https://anthropic.skilljar.com/ai-fluency-for-educators

10. Teaching AI Fluency
   - 说明：帮助教师和课程设计师在课堂中教授和评估 AI Fluency 技能，适合教育工作者开展 AI 教学。
   - 链接：https://anthropic.skilljar.com/teaching-ai-fluency

11. AI Fluency for Nonprofits
   - 说明：专为非营利组织设计，帮助团队高效、伦理地使用 AI，提升组织运营和使命达成能力。
   - 链接：https://anthropic.skilljar.com/ai-fluency-for-nonprofits

12. Claude with Amazon Bedrock
   - 说明：学习在 Amazon Bedrock 上部署和使用 Anthropic Claude 模型，包含企业级集成和最佳实践。
   - 链接：https://anthropic.skilljar.com/claude-in-amazon-bedrock

13. Claude with Google Cloud Vertex AI
   - 说明：全面掌握通过 Google Cloud Vertex AI 使用 Anthropic 模型的完整流程，适合云端企业部署。
   - 链接：https://anthropic.skilljar.com/claude-with-google-vertex
