# VPS 代理搭建提示词

> 来源：https://github.com/lawrencewzen/lawrence-prompt/blob/main/vps-ip-proxy-prompt.md

## 核心功能
通过 VPS 搭建代理服务，通过住宅 IP 出口，支持 Clash/Shadowrocket 订阅。

## 架构
用户设备 (本地客户端拉取订阅) -> VPS (轻量 HTTP 订阅服务 + Gost SS 服务端转发) -> 住宅 IP (Socks5) -> 目标网站

## 执行阶段

### 第一阶段：信息收集
需要提供：
- VPS IP 及 SSH 端口
- VPS root 密码（或 SSH 登录方式）
- 新用户名（默认 proxyuser）
- 住宅 IP 及端口
- 住宅 IP 认证信息（用户名/密码，如有）

### 第二阶段：VPS 基础安全与环境配置
1. 创建新用户并加入 sudo 组
2. SSH 安全配置：配置密钥登录，禁用密码登录
3. 防火墙配置：放行端口
4. 网络优化：开启 BBR

### 第三阶段：核心代理与订阅服务搭建
1. 安装与配置 Gost
2. 生成订阅配置文件（兼容 Clash 和 Shadowrocket）
3. 搭建轻量订阅服务

### 第四阶段：本地端测试与交付
1. 自动唤起客户端并注入订阅
2. 环境变量验证代理出局 IP

## 关键规则
- 禁止伪造执行，必须真实调用终端/SSH
- 关键操作前必须确认
- 每步完成后必须验证
