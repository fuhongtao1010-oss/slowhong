# 股票投资工具清单

> 来源：网络整理
> 日期：2026-03-22

---

## ① 看公告（最重要）
- **巨潮资讯** https://cninfo.com.cn
  - 上市公司一手信息，只看这个就够

## ② 看数据（进阶必备）
- **理杏仁** https://lixinger.com
  - 财报可视化，很适合个人投资者
- **Wind / Choice**
  - 专业选手用的（贵，但确实强）

## ③ 看行情 + 社区
- **同花顺** https://10jqka.com.cn
  - 看盘 + 财报一体
- **雪球** https://xueqiu.com
  - 看观点、看情绪

## ④ 看新闻（别信息滞后）
- **财联社** https://cls.cn
  - 快讯第一手
- **金十数据** https://jin10.com
  - 宏观 + 全球

## ⑤ 盯盘神器（短线用）
- **选股宝** https://xuangubao.cn/dingpan
  - 盘中信息很快

## ⑥ 查估值（防止瞎买）
- **亿牛网** https://eniu.com
  - 查历史PE
- **且慢** https://qieman.com
  - 存笔记

---

## 总结

真正长期在用的，其实就这几类：
1. 公告 → 巨潮资讯
2. 数据 → 理杏仁 / Wind
3. 行情+社区 → 同花顺 + 雪球
4. 新闻 → 财联社 + 金十
5. 盯盘 → 选股宝
6. 估值 → 亿牛网
