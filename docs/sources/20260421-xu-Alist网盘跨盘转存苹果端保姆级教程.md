### 二、苹果系统使用

1、打开上面的链接，打开后面如下
- 第一个是给 intel 芯片的苹果电脑使用的
- 第二个是给 M 系列芯片的苹果电脑使用的
按需下载

2、下载后解压
将解压后的放到一个新建的文件夹中

3、打开苹果电脑终端
"command+空格"打开后搜索"终端"

4、打开终端后
先输入：
```bash
cd 文件夹的绝对地址
```
怎么找地址，触控板两个手指摁下去，点击拷贝

然后在先后输入下面两条：
```bash
# 第一条
xattr -d com.apple.quarantine ./alist
# 第二条
./alist server
```
就能看见启动内容

### 三、登陆管理后台


用户名：admin
密码是：看刚才的黑色命令框，找到 password=xxxxxx，等号后面的一串字符就是密码

一定要保存好密码，如果忘了还要重新装整个

### 四、全网盘互转配置（百度网盘转夸克网盘）

打开浏览器页面，点击"转到储存"
这是因为还没添加网盘，接下来跟着配置就好：

**第一步：添加百度网盘**

1、登录 Alist 管理后台后，找到左侧"存储"选项，点击"添加"

2、按照下面内容填写：
- 驱动：百度网盘
- 显示文件夹名称：/baidu
- 刷新令牌：按照 2026 最新方法一键获取
  - 打开官方获取地址：https://alist.nn.ci/zh/guide/drivers/baidu.html
  - 拉到页面中间，找到"刷新令牌"板块
  - 点击"点击这里"，跳转到百度网盘授权页面
  - 登录自己的百度账号，完成授权，页面会直接弹出你的 refresh_token
  - 完整复制整个 token，不要删减任何字符，粘贴到 Alist 后台

3、所有信息填完，直接点击"保存"，百度网盘就挂载成功了

**夸克网盘**

2、用同样的方法，再次点击【存储】→【添加】：
- 驱动：夸克
- 显示文件夹名称：/quark

**Cookie 获取方法**
- 登录自己的夸克账号，电脑打开夸克网盘网页版：https://pan.quark.cn/
- 然后两个手指头在触控板上双击，然后点击最下面一个"检查"
- 然后点击上面的这个"网络"，类似于 wifi 的图标
- 然后刷新整个网页
- 刷新后，这个页面也会变化
- 左侧的"名称"列，能上下滑动刷新后滑到最上面的，找到这个"list"
- 在这个 list 里，点击右侧的"标头"
- 下滑，滑到 cookie
- 然后复制整段 cookie

把复制的完整 Cookie 粘贴到 Alist 后台，不删任何字符，点击"保存"，夸克网盘也挂载完成

### 五、转存部分

回到 Alist 主页，左侧菜单栏能看到 /baidu（百度网盘）和 /quark（夸克网盘）

点击复制，就能转到要转存的对象了

## 关键注意事项（避坑指南）

- **Alist 窗口不能关，即终端不能关**：list 必须保持运行状态（黑窗口或终端不能关闭），一旦关闭服务就会停止，转存任务会中断。

- **大文件/大文件夹建议分批转存**：几百GB的内容不要一次全选，建议分批操作，避免中途出错或卡住。

- **百度网盘建议开个会员提速**：非会员限速严重，会严重拖慢传输速度。开1天超级会员就能跑满带宽，划算很多。

- **提前检查夸克网盘空间**：转存前一定要确认夸克剩余空间大于你要转的文件总大小，否则会失败。

- **支持断点续传**：如果中途断网、电脑重启或 Alist 意外关闭，重新打开 Alist 后，任务会自动继续，不用从头再来。
