# Obsidian Help Home

## Summary
Obsidian 官方帮助页概述了它作为本地 Markdown 知识库工具的核心能力：创建 vault、笔记链接、插件、Web Clipper、CLI、Publish、Sync 等。这进一步支持了把 Obsidian 作为 LLM Wiki 承载层的判断。

## Key Takeaways
- Obsidian 的基本工作单元是本地 vault 与 markdown note。
- 链接笔记与文件是它的核心能力之一。
- 插件、主题、Web Clipper、CLI 扩展了它的工作流空间。
- 对于 LLM Wiki，Obsidian 同时适合作为浏览器、编辑器和结构可视化工具。
