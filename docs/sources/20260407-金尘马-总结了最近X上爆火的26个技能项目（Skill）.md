## 关键信息
- 这些项目的共同模式是把“人格 / 风格 / 领域知识 / 方法论”包装成 Skill。
- 热点传播的核心不只是工具能力，也包括人格模拟、风格迁移、可复用提示结构与身份扮演。
- 其中“职场、自媒体系列”更接近个人知识资产化与人格蒸馏；“名人系列”更接近思维模型调用；“玄学系列”则把传统术数或宗教角色结构化成可交互形式。
