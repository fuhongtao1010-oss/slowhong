# Qwen3.6-35B-A3B

## 基本信息

- **参数量**：36B（350亿参数）
- **类型**：Image-Text-to-Text（多模态大语言模型）
- **上下文长度**：最高 262,144 tokens（YaRN 可扩展至 100万+）
- **上月下载**：209,112 次

## 核心特性

- **思考模式（Thinking Mode）**：默认先思考再回答，深度推理
- **工具调用能力**：适合构建 Agent 应用
- **超长文本处理**：YaRN 技术扩展
