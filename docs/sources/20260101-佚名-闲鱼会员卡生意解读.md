# 闲鱼会员卡生意解读

> 来源：洋平707 @MYohei707

## 核心观点
闲鱼已成为"互联网黑市"，会员卡是其中一个热门类目。

## 货源来源

### 正规渠道
1. **官方授权代理商** - 从正规平台进货，商品为官方直充，几乎无售后问题
2. **数字权益供货平台** - 整合上千种数字权益商品，可API自动对接发货

### 灰色渠道
3. **积分兑换** - 用信用卡/手机积分兑换会员，边际成本低但库存有限
4. **联合会员拆分** - 购买京东Plus、88VIP等联合会员，拆分附赠权益出售（主流但有争议）

### 非正规渠道
5. **非官方代理** - 从非正规渠道进货，可能混入不明来源货源
6. **做任务/拉新刷单** - 通过APP拉新活动获取免费会员

## 利润空间

| 类型 | 利润率 | 风险 |
|------|--------|------|
| 官方代理直充 | 5-8% | 低 |
| 联合会员拆分 | 较高 | 中 |
| 积分/非正规渠道 | 最高 | 高 |

## 普通人能做吗？

**可以做的准备：**
- 多准备几个手机号，多账号分发
- 学习闲鱼推流技巧
- 多准备几个手机

**建议：**
- 官方代理渠道可以赚零花钱
- 非正规渠道别碰，风险承担不起
- 行业正在向正规代理模式转型
