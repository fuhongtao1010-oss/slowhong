| **月下载量** | 209,112 |

---

## 核心功能特点

### 1. 多模态支持
- **图像理解**：支持图像-文本-文本任务
- **视频理解**：支持视频输入，可配置帧率采样（默认fps=2）
- **长视频处理**：通过配置可支持小时级视频，最长边可达224k视频tokens

### 2. 推理模式
- **Thinking模式（默认）**：模型会先思考再回答
- **Instruct模式（非思考）**：直接回答
- **Preserve Thinking**：保留历史推理痕迹，优化agent场景决策一致性

### 3. 超长上下文
- 原生支持 **262,144 tokens**
- 支持 **YaRN** 扩展到更长上下文（如1,010,000 tokens）

### 4. 工具调用与Agent
- 内置工具调用能力
- 集成 **Qwen-Agent** 框架
- 支持 **Qwen-Code** 终端助手

## API调用示例

### 文本输入
```python
from openai import OpenAI
client = OpenAI()

messages = [{"role": "user", "content": "Type 'I love Qwen3.6' backwards"}]

response = client.chat.completions.create(
    model="Qwen/Qwen3.6-35B-A3B",
    messages=messages,
    max_tokens=81920,
    temperature=1.0,
    top_p=0.95,
    presence_penalty=1.5,
    extra_body={"top_k": 20},
)
```

### 图像输入
```python
messages = [
    {
        "role": "user",
        "content": [
            {"type": "image_url", "image_url": {"url": "https://..."}},
            {"type": "text", "text": "问题描述..."}
        ]
    }
]
```

### 禁用思考模式
```python
extra_body={
    "chat_template_kwargs": {"enable_thinking": False},
}
```

### 保留思考历史
```python
extra_body={
    "chat_template_kwargs": {"preserve_thinking": True},
}
```
