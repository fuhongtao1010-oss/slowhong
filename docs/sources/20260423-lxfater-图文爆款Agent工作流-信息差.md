## 第一步：调研，模仿的是真掏钱的作业

从那张 56 万浏览的图开始拆。你打开这张图，第一反应多半会想：能火 56 万，那提示词一定很复杂吧？

**不，就一张很简单的图片，提示词超级简单。**

更有意思的是，连这条提示词都不是我自己写的。我从来不凭空想写什么图片的提示词，反而我去找出成功的案例去放大。

**怎么模仿？**

不模仿爆款——因为点赞多不等于有人真的要买。很多爆款是情绪梗、争议梗、猎奇梗，看着爽，跟真需求没关系。

**看销量高的图，掏钱买才是真需要。**

打开小红书，按销量排，把前几名的商品图拿下来，然后把这些图扔进 AI Agent，让它反推提示词。

一分钟的事，拿到提示词，稍微改改就能直接用。

所以 56 万流量那张图，我什么都没做！！我只是从需求出发，找对了那个已经解决问题的答案。

但一张图只能抄一次，要做十张、一百张呢？每张都回小红书重新反推一遍？

---

## 第二步：生产内容，一次定制，批量出图

根据前面的步骤，你很快能搞定第一张图。但换个场景，从健身房换到厨房、从学英语换到韩语，又要从头反推一次？

你可能会想：多反推几次呗，反正也不难。但一张图反推一次，十张图反推十次，累加起来，你还是在做一次性劳动。

**真正的答案：写"元提示词"（Meta Prompt），像填空题。**

一个 Meta Prompt 里留好几个空：场景、字、风格、主题。每次要图，只填这几个空就行。

但问题又来了——手工写这种带空位的指令模板，本身就很麻烦。普通人看着 Meta Prompt 就费劲，根本不想写。

**这时候就需要一个好用的工具跳过手写这一步。**

我的做法是：直接跟一个 CREAO Agent 说，"按这个痛点，给我列出要定制化的选项"——然后和它说"随机切换这些痛点给我生成新的提示词，创作一个原创作品"。

就这简单，原来可以这么简单！！
