# 短视频封面标题生成器 — ah-xhs-cover-title

## 摘要

**一句话：** 输入文章/选题，自动识别内容赛道、匹配钩子手法，从85条爆款公式路由生成标题+封面三行文案。

**核心能力：** 每条输出都能追溯到具体钩子手法代号+案例编号，不是凭空编造。

**触发方式：** `/ah-xhs-cover-title` 或自然语言「帮我生成封面」「短视频标题和封面」

---

## 分类体系

### 第一层：钩子手法（8种）

| 代号 | 手法 | 句式特征 |
|------|------|---------|
| A | 建议祈使 | 建议/尽量/千万不要/尽早... |
| B | 顿悟发现 | 我发现/突然发现/突然悟了... |
| C | 反常识对比 | X其实是Y / 看似X实则Y / X代替不了Y |
| D | 如何方法 | 如何/怎么/N步/N个方法... |
| E | 身份圈定 | N岁/N0后/普通人/普通女孩/打工人... |
| F | 数字锚点 | N个/N本/N条/N年/N万... |
| G | 情绪钩子 | 讨厌/焦虑/低谷/害怕/不体面... |
| H | 热点嫁接 | 借势人名/事件/年份节点 |

### 第二层：内容赛道（6条）

| 代号 | 赛道 |
|------|------|
| 1 | 搞钱赚钱 |
| 2 | 自我成长 |
| 3 | 趋势洞察 |
| 4 | 人际关系 |
| 5 | 职场生存 |
| 6 | 心态情绪 |

---

## 封面三行拆法（4种）

1. **结论前置型** — 结论先行，悬念留后
2. **递进悬念型** — 三行层层递进，钩子埋在三行关系里
3. **祈使直给型** — 直接命令/建议，第三行最强
4. **对比撕裂型** — 两极对比，张力最大

## 质量标准

| 检查项 | 标准 |
|--------|------|
| 标题长度 | 15-30字，不超过35字 |
| 封面三行合计 | 不超过25字 |
| 标题和封面关系 | 封面是标题的视觉翻译，不是复制 |
| 钩子可追溯 | 每条对应具体钩子代号+案例编号 |
| 口播友好 | 标题念出来顺口，不拗口 |

---

## 用户反馈迭代

| 反馈 | 处理 |
|------|------|
| 再来几条 | 用剩余未使用的钩子手法补充生成 |
| 风格再激进一点 | 切换到C反常识对比或G情绪钩子 |
| 太夸张了 | 切换到D如何方法或A建议祈使 |
| 换个方向 | 切换赛道重新路由 |
