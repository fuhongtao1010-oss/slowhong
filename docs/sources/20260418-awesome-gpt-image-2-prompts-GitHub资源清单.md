## 模型对比与社区实验（21个案例）

包含 GPT-Image-2 vs Nano Banana 2、Seedream、MAI-Image-2 等模型的对比测试，以及迭代式提示词调校技巧。

来源：@desds1678, @samifox_ai, @HuliJason, @MikelEcheve 等多位创作者

---

## 核心发现

1. **GPT-Image-2 擅长**：人像皮肤纹理、文字渲染（duct-tape-1）、极简海报、多风格转换
2. **弱点**：复杂比例（头身脚）、多物体数量控制
3. **最佳实践**：参考图+导演式提示词 → 10x 控制力
4. **提示词技巧**：迭代调优、多次小幅调整比一次生成更有效
