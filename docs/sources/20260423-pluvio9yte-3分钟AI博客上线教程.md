# 3分钟极速上线个人博客教程

## 摘要

**方法：** 用 CREAO 的 AI Agent，填三个信息（推特链接 + 风格 + Vercel Token），2分钟自动生成并部署个人博客。

**效果：** 自动抓取近50条推文分析风格 → 生成8篇博客文章 + 推文时间线 + About页 → Vercel 自动部署上线。

**核心洞察：** AI 工具的价值在于"一次构建，反复复用"——填表单就能跑，普通人不需要写代码。

---

## 原文

### 痛点

一直有个拖了很久的待办事项：搭一个博客。不是不想做，是每次打开教程就开始头疼——选框架、配环境、写模板、部署域名……光是"选用 Next.js 还是 Astro"这个问题就能让人纠结半小时。最后结果总是一样：关掉标签页，继续刷推特。

### 操作步骤

打开 CREAO，一句话建了一个叫"Twitter Blog Builder & Deployer"的 Agent，填了三个东西：

1. 推特主页链接
2. 博客风格（选了白底极简）
3. Vercel Token（去 vercel.com/account/tokens 点一下生成，粘贴进来）

然后点运行，等了两分多钟，博客就上线了。

### Agent 在这两分钟里做了什么

1. 通过网络搜索抓取了近 50 条推文
2. 分析了话题分布、写作风格、高频词、哪些推文互动最高
3. 根据分析结果，自动生成了 8 篇和风格一致的博客文章
4. 加上推文精选时间线、一个关于"我"的介绍页
5. 用 Vercel CLI 打包部署上去

整个过程没写一行代码，没打开过任何编辑器或终端。

### 效果

博客地址：https://pluvio-blog.vercel.app

白底黑字，极简风格，有点 Linear 官网那种质感。文章卡片点开有完整正文，推文时间线展示了互动数据，About 页展示了技术标签和简介。

不满意风格？直接跟 Agent 说"帮我改成深色风格"，重新生成一版，重新部署，新地址几十秒后出来。

### 为什么值得分享

这个 Agent 是可复用的。它被保存成一个带表单的工具，下次换一个推特账号填进去，走同样的流程，又是一个全新的博客。更新博客也只需要一句话，发给 Agent 就能更新。

朋友想试，直接把表单链接发给他，他填自己的信息，自己的博客就出来了。
