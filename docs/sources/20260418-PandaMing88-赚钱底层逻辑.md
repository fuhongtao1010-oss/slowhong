# 赚钱底层逻辑

## 摘要

文章核心观点：做副业/私域，本质是把行为心理学原理用到真实的人身上，而不是只服务 DAK 数据。

**四大机制：**
1. 变比率强化 — 让用户上瘾的不是固定奖励，是随机奖励
2. 瞭望塔效应 — 用户自我监控，比强制管控更有效
3. 损失厌恶 — 让你困在平台的不是喜欢，是害怕失去已有的
4. 算法读的不是"你喜欢什么"，读的是"你在什么状态下最软"

**AI时代的差距：**
懂原理的人+AI = 效率放大30倍
不懂原理的人+AI = 更快地产出无人问津的内容

**三件武器：**
1. 精准看见 — 算法做不到个体识别，你能
2. 悖论性透明 — 承认你在用什么机制，反而建立最高信任
3. 匹配已有焦虑 — 不制造需求，只在对的时刻出现

## 原文

这一篇聊挣钱的底层逻辑。

我先问你一个问题。

上一次你刷短视频，你打算刷几分钟？

不用回答我。你心里已经有答案了，而且你知道那个答案是假的。

一、你不是用户，你是实验材料

1930年，斯金纳把一只鸽子关进箱子。箱子里有一根杠杆。踩下去，有时候掉出食物，有时候没有，随机的。

结果那只鸽子对那根杠杆产生了近乎宗教式的痴迷，踩的频率，远超固定奖励。

斯金纳把这个叫做"变比率强化"。行为心理学教材里的经典案例。每个认真学过心理学的人都知道这个实验。

然后他们打开手机，向下滑。

这不是在骂谁蠢。我想说的比这残忍得多：人类在这个机制面前，跟鸽子没有任何神经层面上的本质区别。 知道原理，不等于免疫。这是斯金纳之后所有实验的共同结论——被研究得最透的那些认知偏误，往往也是最难逃脱的那些。

所以当字节跳动的产品经理在2016年做出那个"无限下拉"的决策时，他们不是在做产品创新，他们是在把1930年代的笼子，升级成了一个全球规模的、放在所有人口袋里的、每天工作24小时的斯金纳箱。

升级版唯一的改动，是你可以随时离开。

但你没有。

这就是那个设计的真正天才之处——不是让你出不去，而是让你自愿留下来，同时相信你随时可以走。

自由感是留存机制的最后一道涂层。把它磨掉，下面是笼子。

二、他们不需要骗你，他们只需要了解你比你更早

每次你在某个平台上完成一个动作——滑动、停留、回看、截图、快进——那个平台都在记录一件事：在什么情绪状态下，你对什么刺激的抵抗力最低。

不是记录你喜欢什么。是记录你在什么时候最软。

这个区别，比你想象的要大。

你喜欢纪录片。你在知识焦虑的状态下，打开纪录片，看了三分钟，算法切进来，推了一个搞笑视频。你笑了。然后又一个，又一个。四十分钟后你关掉手机，有一种说不清楚的不满足感，像吃了一整袋薯片之后的那种。

你以为算法读错了你。

算法没有读错你。算法读的根本不是"你喜欢什么"，读的是"你在那个特定状态下，会被什么接管"。它知道你想看纪录片，它也知道你在轻度焦虑的情绪基线下，搞笑内容对你的留存效果是纪录片的三倍。

它选了对平台更好的那个，不是对你更好的那个。

而你，觉得那是"你自己的选择"。

你现在有没有想起某一个具体的晚上？

三、不是控制你，是让你控制自己

福柯在1975年描述了一种监狱设计——圆形监狱（Panopticon）。中央一座瞭望塔，四周是牢房。囚犯不知道守卫在不在、在不在看，但他们知道自己随时可能被看见。

结论是什么？

不需要有人真的在看。囚犯会开始自我监控。

福柯说，现代权力的精髓不是压制，而是内化。你把那个瞭望塔搬进自己脑子里，然后替权力打工，不需要任何报酬，还觉得这是自律。

现在来说那些平台。

当你在一个社区里发言时，你会在发送之前在脑子里过一遍：这条内容会不会被推荐？措辞对不对？会不会引战？我的粉丝会怎么看？

你以为这是你在思考。

这是那个瞭望塔。

那个瞭望塔是他们建的。塔里从来没有人。但你一直在表演给那座空塔看。你的内容选题、你的发言频率、你措辞的边界——这些都是那座塔塑造的，你在它塑造的坐标系里活动，还把这叫做"做自己"。

这才是最高级的控制。不需要命令，不需要监控，不需要任何成本。让人自愿成为自己的囚卫。

你现在在想什么样的内容会让这篇文章被更多人分享吗？

我也是。

四、损失厌恶：让你不敢离开的，是你以为自己拥有的东西

卡尼曼证明了一件事：失去一百块的痛苦，大约是得到一百块的快乐的两倍。

这个发现，被互联网平台用到了一个很少被讨论的地方：你的社会身份。

你在某个平台上发了三年的内容。你有粉丝，有互动记录，有那些你已经不记得但偶尔翻出来还会觉得"当时写得不错"的帖子，有人认识你，有人知道你是谁。

这些，不是功能，是资产。但那个资产，账本在平台手里。

哪天平台改规则、改算法、倒闭、封号——那些资产归零。你不是资产的所有者，你是使用者。这一点从来没有人明说过，但写在用户协议的第十七页，字号最小的那行。

所以你不走。不是因为喜欢。是因为不走的损失，感觉上比走要小。

损失厌恶让你困在一个你早就不满意的地方，出不去，也不想出去，然后把这个叫做"积累"。

大多数人的"平台依赖"，本质上是斯德哥尔摩综合征的一个轻量版本。

说出来也不会有人相信。因为承认这一点意味着过去三年的努力需要被重新估值。那个代价太大了。

所以大家继续留着。

五、你现在想做副业，恭喜你，你想成为的那个人

现在来说你。

你做副业，你做私域，你做社群，你在X写内容，你想建立自己的用户群，你想有人付费跟你。

我需要问你一个问题，而且这个问题的答案多数人不愿意直视：

你知不知道，你正在建的东西，跟上面描述的那些机制，用的是同一套底层原理？

变比率强化。你更新的节律，就是那个杠杆的随机性。 社会身份捆绑。你的社群成员积累的参与感、人脉、归属感，是你持有账本的资产。 自我监控内化。你的风格、你的腔调、你的"人设"，会成为成员发言时脑子里的那个瞭望塔。

区别不是你用没用这些东西，区别是你知不知道你在用，以及你用它服务谁。

大厂用这些机制，服务的是KPI报表里那个DAU数字背后的广告主。 你用这些机制，服务的——如果你是认真的——是真实的人，以及你们之间的真实交换。

这是完全不同的伦理，但这是完全相同的心理学。

你想做好副业，你必须先承认这一点，然后才能开始真正地运用它。

六、AI来了，但来的方式让多数人更惨

现在来说AI。

AI没有给所有人平权。这个说法是善意的谎言。

AI做的事情，是把"懂原理的人"和"不懂原理的人"之间的执行效率差距，拉大了大概二十倍。

原来，懂人性的人做副业，比不懂的人，效果可能好三倍，因为执行速度的差距会拖平一部分认知优势。

现在，懂人性的人有了AI，认知优势被放大，执行差距被抹平。那三倍变成了三十倍。

不懂的人有了AI，得到的是：以前一个月写四篇文章，现在可以写四十篇，然后收获四十篇的无人问津。

效率放大的，是已经存在的东西。正确的东西变得更正确，错误的东西变得更错误。

然后是这一段，这是整篇文章最重要的一段，我会用最平静的语气说它：

一个掌握了行为心理学底层原理的人，在AI时代，拥有了和大公司的产品经理+数据团队+运营团队几乎对等的个体工具集。

变比率内容设计——AI可以帮你分析哪种内容节奏让你的受众停留周期最长。 用户情绪地图——AI可以从成员的发言里归纳出他们在哪个情绪状态下最容易被激活。 个性化触达——AI可以帮你对不同特征的成员，在不同节点，说出精准落在他们当下状态的话。 身份镜像设计——AI可以帮你把"你的社群成员是什么样的人"这件事，设计成他们自己愿意认同的形象，而不是你强加的标签。

这些，过去需要一个三人运营团队外加一个心理学顾问。现在需要一个真正理解这些机制的人，以及几个可以复用的提示词工作流。

但注意，前面那句话里最重要的词，不是AI，不是工作流，不是工具集。

是"真正理解"。

AI不会给你那个理解。AI只会把你已有的理解，变成可以大规模执行的东西。

如果你现在对人性的理解是模糊的、直觉性的、凭感觉的——AI给你的将是：更快地做出更多凭感觉的东西，然后把感觉的误差，也一起放大。

七、屠龙者的三件武器，和一个不那么好听的真相

武器一：让你的人感到被"精准看见"

人对"被看见"这件事的渴望，被几十年的内容工业系统性地低估了。

大平台的算法给人的是被"类型化"——你被归入一个用户标签，然后喂给你这个标签对应的内容。效率很高，但那个感觉，像是在一家便利店被店员递给你"你们这类人喜欢的东西"。

个体做私域的真实优势，不是"比大厂更走心"这种模糊的情感描述，而是一个非常具体的能力：你能做到算法做不到的个体识别。

一个新成员进入你的场域，你（或者你的AI工作流）能在最短时间内，从他的发言、提问方式、在意什么、回避什么里，拼出一张他自己未必清楚的图——然后你回应他的，是这张图，不是他所在的那个用户标签。

这个差距，就是护城河。大厂不是不想做，是因为他们的规模，让这件事不可能发生在每个用户身上。你的规模小，这是你唯一的真实优势。不要用批量化操作把它扔掉。

武器二：悖论性透明——说出那些"不该说"的话

大平台有一件事永远不能做：承认自己在用这些机制。

你可以。

你说"我知道我在设计内容节奏是为了让你们更稳定地留下来，我在用心理学，我会告诉你我怎么用的"——这句话，在大厂嘴里是公关危机，在你嘴里是建立最高密度信任的核弹。

这叫悖论性透明。越坦白你的操作原理，越让人觉得你是唯一一个不会操控他们的人。

这个逻辑是真实的，不是话术。因为真正的操控需要隐藏，而把机制暴露出来的人，在那个暴露的瞬间，已经放弃了这个机制最核心的武器。

这是个体对组织的少数天然优势之一，大多数人没有用，因为他们自己也没有把这些想清楚。

武器三：不要制造焦虑，要精准匹配已经存在的焦虑

大厂的常规路径，是制造需求。让你觉得你有一个你本来没有的问题，然后卖给你解决方案。

这条路的天花板是你有多少媒介预算，以及你愿意对用户造成多大的心理伤害。

个体做私域有一条不需要制造任何东西的路：你的用户已经有焦虑，你只需要成为那个在正确时刻出现的人。

AI可以帮你建一张你的用户群体的情绪地图——他们反复出现的困扰是什么，在什么情境下被触发，在什么时间段强度最高。这不是为了操控，是为了让你出现的时机，和他们需要的时机重合。

时机比内容质量更重要。一句普通的话，在对的时刻说出来，会被当成一句改变人生的话记住。一篇好文章，在对的人情绪关闭的时候推出去，石沉大海。

大厂花了几十年把这件事做成了算法。你用AI工作流，用更小的规模，用更人性化的感知，可以做到一个个体应该能做到的精度。

八、最后，关于那条龙

所谓暗中屠龙，龙不是某家公司，不是某个算法，不是某个产品经理。

龙是一套关于人类注意力和情感脆弱性的操控架构，它在过去二十年里被工程化、规模化、产品化，最终变成了你每天早晨醒来摸的那块屏幕。

大多数做副业的人，是在龙肚子里建一个小窝，以为这叫"独立"。

真正的独立，是先爬出来，站在外面，彻底搞清楚那条龙是怎么运作的，然后用同样的原理，建一个真正属于你的、以人为核心的、AI可以帮你执行的东西。

这需要你承认一件不太舒服的事：你现在的用户，和当初把你变成斯金纳箱里那只鸽子的那些用户，是同一类人，也包括你自己。

承认这一点，不是自我攻击。是进入真正操盘者视角的唯一入口。

你对他们充满理解，因为你就是他们。 你对他们充满善意，因为你知道那个笼子是什么感觉。 你用同样的机制，但你不把那个机制藏起来，你把它用来服务真实的交换。

这就是降维打击的真实含义——不是你比别人聪明，是你比别人对人性更诚实，对自己更诚实。

主业养家糊口，副业暗中屠龙。

屠的，首先是你自己脑子里那条龙。

其他的，会跟着来。
