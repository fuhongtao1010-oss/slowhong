# 2026-04-08 - 努力赚钱的菜狗 - GitHub上8个高星的AI短剧制作项目

## 摘要
这是一条围绕“AI 短剧”赛道的工具整理帖。作者认为 AI 小短剧是当前很热、很容易商业化的方向，并列出 8 个 GitHub 上的高热项目，覆盖小说转短剧、一句话生成完整短剧、工业级影视自动化、短视频引擎、Novel2Video 等方向。

## 原文整理
昨天刷了一天抖音 AI 小短剧，上头了，还充了 30 块钱。

了解了一下，AI 短剧确实是当下最火最赚钱的渠道，爆款单剧能拿到几十万，各大平台补贴超百亿。

我整理了 GitHub 上 8 个高星的 AI 短剧制作项目，利用这些工具就可以一键式生成 AI 短剧，上传各大短视频平台赚钱。

1. Toonflow
- 小说自动转短剧、漫剧
- 仓库： https://github.com/HBAI-Ltd/Toonflow-app

2. 火宝短剧
- 一句话生成完整短剧，从剧本到成片全自动化
- 仓库： https://github.com/chatfire-AI/huobao-drama

3. waoowaoo
- 工业级 AI 影视全自动生成平台
- 从小说自动策划分镜、角色、视频与配音
- 仓库： https://github.com/saturndec/waoowaoo

4. Jellyfish
- 一站式短剧生成工具
- 从剧本到分镜，再到一致性管理与后期剪辑
- 仓库： https://github.com/Forget-C/Jellyfish

5. MemeCalculate / moyin-creator（链接文字在帖文中有截断）
- AI 影视级别生产工具
- 支持 Seedance 2.0
- 剧本到成片全流程自动化
- 大概率相关仓库： https://github.com/MemeCalculate/moyin-creator

6. AIDC-AI / Pixella
- AI 自动短视频生成引擎
- 自动写文案、配乐、视频等
- 支持漫画风格，零门槛
- 仓库： https://github.com/AIDC-AI/Pixella

7. Bigbanana
- 一句话生成完整短剧动漫
- 一致性强，工业化生成首选
- 仓库： https://github.com/shuyu-labs/Bigbanana

8. ViMax
- 内置 Novel2Video 模块
- 专为长篇小说设计
- 仓库： https://github.com/HKUDS/ViMax

作者建议：
- 想用导入小说、书本生成短剧的，可以优先尝试 Toonflow、火宝、waoowaoo。

## 关键信息
- 这条帖子把 AI 短剧赛道描述成“高热且可商业化”的方向。
- 工具形态主要集中在三种：
  - 小说 / 网文直转短剧
  - 一句话 / 主题直出完整成片
  - 工业级批量化影视生产
- 作者推荐优先看 Toonflow、火宝和 waoowaoo，这几项更适合从文本直接进入短剧生产。
