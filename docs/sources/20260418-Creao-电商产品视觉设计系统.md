# 电商产品视觉设计系统

## 摘要

Creao 出品的电商产品视觉设计系统，是一套基于 6 角色协同的 AI 提示词体系。通过品牌策略总监、视觉设计总监、产品摄影专家、文案策划专家、场景应用设计师、技术规格管理六大角色分工，为任意产品类别生成完整的电商视觉素材包（7 个标准模块）。

## 系统架构

### 六大核心角色

| 角色 | 职责 |
|------|------|
| 品牌策略总监 | 品牌调性、目标用户、核心卖点 |
| 视觉设计总监 | 视觉规范、版式系统、配色方案 |
| 产品摄影专家 | 拍摄角度、光影方案、构图细节 |
| 文案策划专家 | 卖点文案、场景描述、行动召唤 |
| 场景应用设计师 | 3-4 个典型使用场景卡片 |
| 技术规格管理 | 规格参数、物流信息、信任背书 |

### 输出清单（7 个标准模块）

1. **主视觉海报** — 产品 + 品牌 + 核心卖点 + CTA
2. **产品卖点卡片组** — 3-4 张，每张一个核心特性
3. **产品细节特写** — 切面/纹理/质感展示
4. **场景应用卡片组** — 3-4 个典型使用场景
5. **产品规格说明** — 重量/尺寸/包装/储存
6. **物流保障模块** — 包装展示 + 信任标签
7. **行动召唤设计** — 购买按钮 + 优惠信息

### 视觉风格规范

```
背景色：#F5F1E8 至 #FDFBF7（米色/奶油色系）
主标题：36-48pt 粗体深色
副标题：18-24pt 常规灰色
标签：圆角矩形，半透明白底
摄影：柔和自然光，45 度侧光
```

### 工作流程

```
品牌策略总监（定调）→ 视觉设计总监（规范）
→ 各角色并行创作（摄影/文案/场景）
→ 技术规格管理（整合输出）
```

## 使用方法

**完整提示词**（见正文）发给 AI 后，按以下格式输入产品信息：

```
产品名称：
产品类别：
核心特性（3-5 个）：
目标用户：
价格区间：
```

系统即自动生成完整的 7 模块视觉设计包。

## 适用产品类别

- 生鲜食品（视觉重点：新鲜度、质感；文案侧重：口感描述、营养价值）
- 数码产品（科技感、细节；功能特性、技术参数）
- 家居用品（质感、环境融合；舒适度、实用性）
- 服饰鞋包（材质、搭配；风格定位、穿搭场景）
- 美妆护肤（质地、效果；功效承诺、使用感受）

## 我的备注

- 这套提示词本身不直接生成图片，只输出设计规范 + 文案内容
- 最终图片需配合 Figma/Canva/Midjourney 等工具制作
- 提示词可存为 Skill 模板，便于下次快速调用
- 平台适配：淘宝（信息密度高）、京东（简洁专业）、拼多多（价格冲击）、小红书/抖音（生活方式化）
