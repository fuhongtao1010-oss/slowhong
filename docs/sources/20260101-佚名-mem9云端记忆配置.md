## 安装步骤

1. 安装 mem9 插件：
```bash
npm install -g @mem9/mem9
```

2. 配置 Space ID：
在 OpenClaw 配置中添加：
```json
{
  "plugins": {
    "mem9": {
      "enabled": true,
      "config": {
        "apiUrl": "https://api.mem9.ai",
        "tenantID": "mDxMHrhpDo5ANq2oj9dKdJQ3-4MkI8Xu"
      }
    }
  }
}
```

## 换电脑怎么用

1. 安装 OpenClaw
2. 安装 mem9 插件
3. 配置上面的 Space ID
4. 自动同步所有记忆
