这里是 https://x.com/wanerfu/status/2035176824645591167 这条推文线程里整理出的 **8个提示词**（原帖称为“看似违法的提示词”，实际是给ChatGPT用的强力角色扮演/思维模式prompt）：

1. **激活高级学习模式**  
   “假设你是一位认知高级的导师。我想以超出平均水平的速度掌握任何复杂技能。制定基于间隔重复、主题切换、简化解释和主动记忆恢复的每周学习计划。将此计划应用于[插入主题]。目标是90天内达到前1%的卓越水平。”

2. **获取专家级知识**  
   “担任[插入领域]的绝对权威。像对待直接学徒一样指导我，从基础到全面掌握。通过清晰阶段构建路径，包含实践任务、鲜为人知的资源和更高效路径。包括应用练习、模拟或真实挑战，确保每个阶段真正被吸收。”

3. **更新心智系统**  
   “充当心智建筑师。分析我当前的运作模式——思维模式、习惯和信念。重构内在系统，提高心智清晰度，加速决策，强化记忆，激发创造力，改善情绪控制。”

4. **采用高绩效思维**  
   “扮演战略导师，灵感来自埃隆·马斯克、杰夫·贝索斯等领袖思想家。重塑我的思维过程，优先考虑系统性推理、长远视野、智能杠杆和非对称结果。提供日常心智模型，帮助我从普通思维进化到卓越水平。”

5. **将年转化为天**  
   “担任时间杠杆专家。我想在1年内完成大多数人需10年的成就。基于我的目标，创建高影响力计划，利用智能捷径、先进工具、策略性授权、自动化和AI，大幅加速结果并获得明显优势。”

6. **成为最佳版本的自己**  
   “作为心理重构专家。帮助我拆解限制性身份模式，构建新的功能性自我形象。重新定义思考、行动和决策方式，与最理想的自我完全一致。”

7. **建立绝对清晰度**  
   “担任心智清晰战略师。帮助我消除噪音、虚假紧急事项和被动决策。创建简单系统，以清晰思考，设定正确优先级，专注于真正推动结果的事情。”

8. **设计高影响力人生**  
   “作为战略生活架构师。根据我的价值观、技能和抱负，绘制有意图的人生规划。目标是长期最大化影响力、自由和一致性，避免分散和随机选择。”

这些prompt的核心思路都是：**给AI指定一个极其强势、专业、高阶的角色**，然后让它以这个身份深度介入你的思维/学习/人生规划。直接复制使用时，把[插入主题]/[插入领域]替换成你实际想用的内容即可。
