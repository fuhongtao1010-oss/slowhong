# 2026-04-07 - 歸藏 - 整理一下藏师傅开源的所有 Skill

## 摘要
这是一条来自 X 的整理帖，汇总了 op7418 公开发布的 6 个 Skills / 工具仓库，覆盖 IM 远程控制、视频包装、YouTube 长剪短、文档配图、中文去 AI 味、PPT 图片与视频生成等方向。

## 原文整理
整理一下藏师傅开源的所有 Skill

Claude-to-IM-skill：
约 1800+ stars
安装后可以将你的 Claude Code 和 Codex 连接到任何聊天软件，实现远程控制
仓库： https://github.com/op7418/Claude-to-IM-skill

Video-Wrapper-Skills：
约 200+ stars
自动为你的视频添加类似“小Lin说”那样的解释特效，支持高亮卡片、关键词卡片、作者介绍、社交媒体卡片等多种组件
仓库： https://github.com/op7418/Video-Wrapper-Skills

Youtube-clipper-skill：
约 1700+ stars
帮你从 YouTube 下载长视频，并在分析视频后给出“长剪短”的方案，同时会自动添加中英双语字幕。
仓库： https://github.com/op7418/Youtube-clipper-skill

Document-illustrator-skill：
约 300+ stars
自动分析文档并生成配图，支持 16:9 和 4:3 比例。它会按照你的分段生成多张配图，并支持多种风格选择。
仓库： https://github.com/op7418/Document-illustrator-skill

Humanizer-zh：
约 5600+ stars
去除 AI 生成文档中常见的“AI 味”表述，采用两段式处理，帮助提升 AI 生成内容的自然度。
仓库： https://github.com/op7418/Humanizer-zh

NanoBanana-PPT-Skills：
约 2100+ stars
基于 AI 自动生成高质量 PPT 图片和视频，支持智能转场和交互式播放。
仓库： https://github.com/op7418/NanoBanana-PPT-Skills

## 关键信息
- 这组仓库大致分成三类：
  - 基础设施 / Agent 接入：Claude-to-IM-skill
  - 内容处理 / 润色：Humanizer-zh
  - 多媒体生成：Video-Wrapper-Skills、Youtube-clipper-skill、Document-illustrator-skill、NanoBanana-PPT-Skills
- 从星数与传播度看，Humanizer-zh、NanoBanana-PPT-Skills、Claude-to-IM-skill 是最值得优先关注的几个。
