# 飞书学习系统使用指南

## 系统架构

```
飞书学习系统/
├── prompts.txt          # 核心提示词
├── 使用说明.md           # 本文件
└── 模板/
    ├── 每日技能打卡.md   # 每日学习记录
    ├── 知识库整理.md    # 飞书知识库模板
    └── 项目实战.md      # 实战项目记录
```

---

## 快速开始

### 方式一：直接对话（推荐）

直接在 Claude Code 中告诉我你想做什么：

| 你说 | 我会做 |
|------|--------|
| "飞书今天学什么" | 推荐今日学习内容 |
| "飞书打卡" | 记录今日学习成果 |
| "飞书实战" | 分配一个实战任务 |
| "飞书知识库" | 帮你整理知识体系 |
| "飞书周报" | 总结本周学习进展 |

---

### 方式二：模板记录

1. 打开 `模板/每日技能打卡.md` → 记录当天学习的技能
2. 打开 `模板/项目实战.md` → 记录实战项目进展
3. 打开 `模板/知识库整理.md` → 构建个人知识体系

## 飞书核心技能清单

### 入门必学（1-2周）
- [ ] 文档基础：编辑、排版、插入内容
- [ ] 云文档：多人协作、评论、点赞
- [ ] 快捷键：提升操作效率
- [ ] 基础搜索：快速找到内容

### 进阶技能（3-4周）
- [ ] 多维表格：数据管理与可视化
- [ ] 智能助手：AI辅助内容创作
- [ ] 流程助手：审批与自动化
- [ ] 飞书日历：日程管理与会议

### 高阶技能（5-8周）
- [ ] 飞书机器人：消息推送与自动回复
- [ ] API集成：与其他工具打通
- [ ] 企业知识库：搭建团队知识中心
- [ ] 自动化工作流：效率飞跃

## 常见问题

**Q: 每天要学多久？**
A: 建议15-30分钟，用番茄钟法更高效。

**Q: 之前没用过飞书怎么办？**
A: 从"入门必学"开始，我会在"今天学什么"中给你安排。

**Q: 学完做什么？**
A: 可以在工作中应用，也可以记录成知识库分享给他人。
