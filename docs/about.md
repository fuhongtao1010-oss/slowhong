# 关于我

你好，这里是 `slowhong`。

这个网站主要记录四类内容：

- AI 工具和使用方法
- 建站、发布和个人内容系统
- 效率工具、工作流和日常折腾
- 一些值得长期保留的文字内容

## 为什么要做这个网站

平台内容适合传播，但不适合承载全部内容资产。很多值得留下的文章、笔记和经验，最好还是放回自己的站点里。

这个网站的目标很简单：

- 把内容留在自己可控的地方
- 建一个长期可维护的发布入口
- 一边写，一边逐步优化结构和体验

## 目前的写作方向

### AI 与工具

记录日常使用过、踩过坑、真正能提效的工具和方法。

### 建站与发布

包括 Markdown、GitHub、Cloudflare Pages、静态网站和低成本搭建方案。

### 长期写作

把零散经验整理成系统内容，不只发即时动态，也尽量沉淀可复用的文章。
